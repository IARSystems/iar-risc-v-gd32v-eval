Example description

  
   

  
  • Introduction
    • 

  This project provides a clean slate as a shortcut for starting new projects with the IAR GD32V Eval Board and the IAR Embedded Workbench for RISC-V.

  This template can be used for quickly creating new projects and also for contributing with new projects on the contributions folder in the official repository.

  
  • Details of the Template
    • 

  - It provides GD32VF103 MCU system clock setup to 108 MHz.

  - All the drivers from the GD32VF103 Firmware Library are included by default.

  - All the BSP drivers for the IAR GD32V Eval Bbard are included by default.
  
  
  • How to use it ?
    • 

  In order to make the program work, you must do the following :
  - Rebuild all files and load your image into target memory
  - Run the example

Template description

Introduction

This template project provides a clean slate for starting new projects with the IAR RISC-V GD32V Eval board on the IAR Embedded Workbench for RISC-V.

It can be used for quickly creating new projects using the GD32VF103 Firmware Library.

It also can be used for contributing with new projects on the contributions folder in the board's repository. Visit https://github.com/IARSystems/iar-risc-v-gd32v-eval/tree/master/contributions if you would like to know more.

Details of the Template

  • It provides GD32VF103 MCU system clock setup to 108 MHz.
  • All the drivers from the GD32VF103 Firmware Library are included by default.
  • All the BSP drivers for the IAR RISC-V GD32V Eval board are included by default.

    • How to use it?

  • Rebuild the project: Project / Rebuild All.
  • Flash the ELF image into the target's memory and start debugging: Project / Download and Debug (CTRL+D).
  • Run the project: Debug / Go (F5).
  • Terminate the C-SPY debugging session: Debug / Stop Debugging (CTRL+SHIFT+D).
  • Develop the desired firmware application.

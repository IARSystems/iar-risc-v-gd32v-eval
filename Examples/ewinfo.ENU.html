Example descriptions

The following project examples use the GigaDevice GD32VF103 Firmware Library. They are provided for the IAR GD32V RISC-V Eval board.

  • accelerometer
    Exercises the MMA8652 I2C 3-axis accelerometer mounted on this board.
  • coremark
    CoreMark is a benchmark that measures the performance of the combination of CPU and compiler. It is very popular for embedded systems.
  • leds
    Blinks the board LEDs.
  • potentiometer
    Uses the ADC to read the potentiometer value.
  • rbg-led
    Pulses the board RGB LED.
  • spi-flash
    Exercises the board SPI Flash.
  • switches
    This example performs polling to check whenever the switches S1, S2 or S3 are pressed, then the corresponding LED is toggled.
  • switches-interrupt
    This example uses EXTI interrupts to check whenever the switches S1, S2 or S3 are pressed, then the corresponding LED toggles.
  • timer-pwm
    Demonstrates how the TIMER peripheral PWM configuration can be used to pulse the RGB LED.
  • temperature-humidity
    This project exercises the SHT31 I2C Temperature/Humidity sensor mounted on this board.
  • toggle-io
    This project is an example which toggles an LED available on a GPIO pin.
  • usart
    This example performs interrupt-driven USART communication.
